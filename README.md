# Fruit Classifier

### Getting started

- Clone this repo
- Install requirements
- Run the `python app.py`
- Check http://localhost:5000

### Install requirements

```shell
$ pip install -r requirements.txt
```

Make sure you have the following installed:
- tensorflow
- keras
- flask
- pillow
- h5py
- Jinja2

### Run the app

```
$ python app.py
```
